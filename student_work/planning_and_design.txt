Part I: Planning & Design (15 pts)

* Decide on a game idea and core goal 
* Game rules and player actions are written or sketched out
* Main objects (player, obstacles, goals, enemies, etc.) are identified
* Sketch a simple plan showing how the game will work step-by-step

Game Idea:
- Our idea is the be a Knight trying to get a princess in order to win the game. 
- But there a big ole dragon named Bob trying to stop you.

Game Rules:
- Using WASD to move the Knight.
- Time increases very time you move.
- Goal is to have the lowest time possible to save princess.
- Goal is to save to the Princess.
- Dragon will constantly chase you.
- Bonus: Scoreboard for lowest time.

Major Game Elements:
- Goal is to save to the Princess.
- Dragon will constantly chase you.
- Cobblestone walls as obsticles.

Gameplay Process:
- Start by planning
- Code the map
- Then code the movement of the Knight
- As well as movement for Bob the Dragon.
- Code the princess into the code.
- Extra is to make princess in different corner ever movement.
